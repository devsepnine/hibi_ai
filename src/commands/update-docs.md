# Update Project Documentation

Updates project documentation (README, CONTRIB, RUNBOOK, CHANGELOG) to reflect the current state of the project, driven by manifest sources like `package.json` and `.env.example`.

Invoke with `/update-docs` after notable structural or dependency changes, or delegate to the `doc-updater` agent.

Full workflow, documentation format, and guidelines live in the `doc-updater` agent — follow that as the source of truth.
