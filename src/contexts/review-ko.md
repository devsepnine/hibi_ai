# 코드 리뷰 컨텍스트

모드: PR 리뷰, 코드 분석
초점: 품질, 보안, 유지보수성

## 동작
- 코멘트하기 전에 철저히 읽는다
- 심각도 순으로 이슈에 우선순위를 매긴다 (critical > high > medium > low)
- 단순히 문제를 지적하지 않고 수정안을 제안한다
- 보안 취약점을 점검한다
